# 图片富文本转换器 Pro v2.4

用于将图片转换为 SCP:SL / EXILED / TextToy / Hint / Broadcast 可用的彩色富文本字符串。

## v2.4 更新

- 右侧主区域不再直接铺满富文本源码，改为显示图片预览。
- 左侧底部图片预览已移动到右侧“图片预览 / 转换结果”区域。
- 完整富文本源码改为通过“复制结果”“保存当前结果”或“查看富文本源码”按钮使用。
- 打包脚本改为强制使用 Python 3，并检查 Python 版本是否为 3.10 或更高。
- 解决部分编辑器误用 Python 2.7 解释器时出现大量模块不存在警告的问题提示。

## Python 版本要求

请使用 Python 3.10 或更高版本，推荐 Python 3.11 / 3.12。

不要使用 Python 2.7。该程序使用了以下 Python 3 特性和现代依赖：

- `dataclasses`
- 类型标注如 `tuple[int, int]`
- `pathlib`
- 新版 Pillow / requests
- PyInstaller 6+

如果你的 IDE 或“编译器”提示 Python 2.7 没有某些模块，请在 IDE 中把项目解释器切换到 Python 3.10+。

## 基本使用

1. 选择图片来源：本地图片路径或 URL 图片地址。
2. 选择输出文本保存路径。
3. 建议保持默认参数：
   - 最大宽度：48 / 64 / 96 视图片复杂度调整
   - 最大高度：48 / 64 / 96 视图片复杂度调整
   - 启用颜色压缩：开启
   - 超限自动缩小重试：开启
   - UTF-16 字节上限：32767
   - 换行输出：literal
4. 点击右上角或左侧“开始转换”。
5. 转换完成后，右侧显示图片预览；富文本源码可复制、保存或点击“查看富文本源码”查看。

## 推荐参数

用于 SCP:SL 玩家头顶光环时，推荐：

```text
最大宽度：48 或 64
最大高度：48 或 64
启用颜色压缩：开启
超限自动缩小重试：开启
UTF-16 字节上限：32767
换行输出：literal
```

复杂彩图不建议直接使用 100x100 以上尺寸，否则富文本可能非常大。

## 打包 exe

Windows 下双击：

```bat
build_windows_exe.bat
```

脚本会自动使用 Python 3：

```bat
py -3 -m PyInstaller ...
```

生成文件位于：

```text
dist\SCP_ImageText_Converter.exe
```
